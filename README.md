# Installation Guide

Community-maintained distribution of Professional Orthodontic Design software for Windows and macOS. Use the following commands to install and update via your preferred package manager.

## Installation

### Homebrew (macOS)

To install via Homebrew, first tap the community repository and then install the desired edition:

```bash
# Add the tap
brew tap af-community-maintainer/tap

# Install Standard Edition
brew install archform

# Install CNC Edition
brew install archform-cnc
```

### Winget (Windows)

Search and install using the Windows Package Manager:

```bash
# Install Standard Edition
winget install ArchForm.ArchForm

# Install CNC Edition
winget install ArchForm.CNC

# Install DPA Edition
winget install ArchForm.DPA
```

### Chocolatey (Windows)

Install via the Chocolatey community repository:

```bash
# Install Standard Edition
choco install archform

# Install CNC Edition
choco install archform-cnc

# Install DPA Edition
choco install archform-dpa
```
